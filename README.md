# Task Tracker

A full-stack task management application that allows users to organize, prioritize, and track their daily work.  
Built with a modern React frontend, secure Node.js backend, and PostgreSQL database.

---

## Key Features

- User signup and login (JWT authentication)
- Create, update, and delete tasks
- Task status and priority management
- Overdue task alerts
- Productivity summary and streak tracking
- Clean, responsive dashboard UI

---

## Tech Stack

**Frontend**
- React
- Axios
- Tailwind CSS

**Backend**
- Node.js
- Express.js
- PostgreSQL
- JWT Authentication
